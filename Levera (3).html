Дети **не могут** самостоятельно регистрироваться в Приложении. Все данные о ребёнке собираются исключительно с явного согласия родителя. --- ## 3. Особые условия для данных детей Это самый важный раздел Политики. ### 4.1 Принцип минимизации данных Мы собираем **минимально необходимый объём** данных о ребёнке для функционирования Приложения. Мы не просим настоящее имя ребёнка — только отображаемое. Мы не просим точную дату рождения — только возраст. ### 4.2 Согласие родителя Все данные о ребёнке собираются только с явного согласия родителя или законного представителя. Родитель: - Создаёт профиль ребёнка - Даёт согласие на обработку данных ребёнка при регистрации - В любой момент может удалить профиль ребёнка и все связанные данные ### 4.3 Что не делается с данными детей - Данные о поведении ребёнка **не передаются третьим лицам** ни в каком виде - Данные **не используются** для таргетированной рекламы - Данные **не продаются** никому и никогда - Данные **не используются** для обучения AI-моделей без явного согласия - Внутри Приложения **нет рекламы** для детей ### 4.4 Архитектура хранения данных детей ``` Поведенческие данные ребёнка (skill scores, история квестов, XP, стрик) → хранятся ТОЛЬКО на устройстве. На сервер (Supabase) уходят: - Анонимизированные агрегаты для статистики продукта (например: «средний completion rate по возрастной группе») - Данные для синхронизации между устройствами родителя и ребёнка (зашифровано при передаче и хранении, TLS 1.3 + AES-256) Идентификатор пользователя на сервере ≠ реальная личность. Связь между анонимным ID и личностью хранится только на устройстве. ``` ### 4.5 Соответствие COPPA (для пользователей из США) Приложение **не предназначено** для детей младше 13 лет как самостоятельных пользователей. Ребёнок использует Приложение в рамках семейного профиля, созданного совершеннолетним родителем. Все данные ребёнка обрабатываются с явного согласия родителя в соответствии с требованиями COPPA (Children's Online Privacy Protection Act). Если вы считаете, что ребёнок создал аккаунт самостоятельно без вашего согласия — напишите на privacy@levera.app. Мы удалим данные в течение 72 часов. ### 4.6 Соответствие GDPR-K (для пользователей из ЕС) Для пользователей из ЕС обработка данных детей осуществляется на основании явного согласия родителя (Статья 6(1)(а) и Статья 8 GDPR). Минимальный возраст самостоятельного согласия — 16 лет (или ниже, если национальное законодательство государства-члена ЕС устанавливает иное). --- ## 5.
