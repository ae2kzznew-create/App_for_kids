Пять минут на старте. ``` **Шаг 1:** ``` Скажи, что хочешь развить Говори голосом или пиши. «Хочу, чтобы сам убирал за собой». «Боится ответственности». «Не умеет планировать время». Механика работает с 7 до 14, но лучше всего — в этом диапазоне. Дети старше 13 часто предпочитают меньше «игровых» механик — мы учитываем это в настройках. ``` **В: Ребёнок не захочет пользоваться — как его уговорить?** ``` Не уговаривайте. Первый шаг — ребёнок сам составляет список желаний. Он говорит, чего хочет. Барабан выдаёт из его списка. Это не задание от родителя. Это его игра. Большинство детей заходят сами после первого барабана. ``` **В: Чем это отличается от других приложений с заданиями?** ``` Все остальные построены на логике: задание → монетка. Когда монетки убираешь — поведение исчезает. Levera построен так, чтобы навык остался после приложения. Система вознаграждения постепенно убирает себя. Ни одно другое приложение не делает этого. ``` **В: Что, если ребёнок начнёт и бросит через неделю?** ``` Такое бывает. Вот что делать: Посмотрите на квесты — они подходят по сложности? Слишком лёгкие → скука. Слишком сложные → фрустрация. Посмотрите на список желаний — там есть что-то реально важное для ребёнка? Если нет — обновите список вместе. И напишите нам — мы разберём конкретную ситуацию. ``` **В: Это безопасно? Что происходит с данными ребёнка?** ``` Данные о поведении ребёнка остаются на устройстве. На сервер уходят только анонимизированные агрегаты. Мы не продаём данные. Мы не показываем рекламу внутри приложения. Levera монетизируется только через подписку родителя. ``` **В: Можно ли отменить подписку?** ``` Да. В любой момент — прямо в настройках приложения. Без форм, без звонков, без «вы уверены?» на пяти экранах. ``` **В: AI генерирует задания — это не опасно?** ``` AI генерирует предложения. Родитель одобряет каждое. Ни одно задание не попадает к ребёнку без твоего тапа. ``` --- ## [10] CTA Footer ### Задача блока Финальный призыв.
