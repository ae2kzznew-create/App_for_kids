# Levera — Wizard-of-Oz Операционный гайд *Версия 1.0 — Как вести ручной тест с первыми семьями без AI-кода* --- > **Что такое Wizard-of-Oz тест** > > Ребёнок за шторой крутит колёса. Зритель думает, что работает машина. > > Применительно к Levera: основатель вручную выполняет роль AI — генерирует квесты, отправляет отчёты, ведёт прогресс. Семья думает, что работает приложение. Цель — подтвердить W4 retention **до** того, как написана хотя бы одна строка кода AI. > > Это не обман. Это стандартная практика проверки гипотез. Zappos так продавал обувь. Airbnb так фотографировал квартиры. Levera так проверяет педагогическую механику. --- ## Что тестируем Одну вещь: **работает ли механика квест → ключик → барабан → приз на удержание семьи 4 недели**. Не AI. Не дизайн. Не код. Только механика. Если механика удерживает семью — AI и красивый интерфейс это усилят. Если механика не удерживает — никакой AI это не исправит. --- ## Инструменты теста Всё, что нужно для старта — уже есть на телефоне. | Инструмент | Для чего | Стоимость | |---|---|---| | **Telegram-бот** (основатель вручную) | Канал коммуникации с семьёй | ₽0 | | **Google Sheets** | Трекинг прогресса каждой семьи | ₽0 | | **Figma** | Макеты экранов для демонстрации | ₽0 (бесплатный план) | | **Notion / Google Docs** | Шаблоны квестов, банк желаний | ₽0 | | **Loom или обычное видео** | Weekly Report в формате короткого видео (если нет приложения) | ₽0 | **Опционально (если нужен «настоящий» вид):** - Телеграм-бот с кастомным именем (BotFather, 10 минут настройки) - Простая веб-страница на Tilda/Notion как «приложение» --- ## Структура теста ### До старта (День 0) **Шаг 1 — Онбординг звонок (30 минут)** Позвонить каждой семье. Провести по сценарию: ``` 1. Объяснить механику (10 мин): «Ребёнок будет получать квесты каждый день. Выполнил → нажимает «Готово» в Telegram. Ты подтверждаешь → он получает ключик. Накопил 5 ключиков → крутит барабан → получает приз.» 2. Составить список желаний ребёнка (10 мин): Попросить родителя позвать ребёнка. Ребёнок называет 5–10 желаний голосом или текстом. Родитель для каждого: ✅/❌, редкость (обычный/редкий/супер-редкий). Записать в Google Sheet семьи. 3. Узнать цель развития (5 мин): «Что хочешь развить в [имени]?» Записать дословно. Это основа для первых квестов. 4. Технический старт (5 мин): Добавить семью в Telegram-группу или создать личный чат. Отправить первый квест. ``` **Шаг 2 — Создать Google Sheet для семьи** Шаблон (одна строка = один день): ``` | Дата | Квест | Статус | Ключики | Стрик | Заметка | |------|-------|--------|---------|-------|---------| ``` **Шаг 3 — Подготовить первые 5–7 квестов** Вручную, на основе цели родителя. Использовать ZPD-калибровку: - Если навык новый → L3 (подробная инструкция) - Если что-то знакомое → L2 (чеклист) - Если умеет, но не делает регулярно → L1 (только цель) *Шаблоны квестов — в следующем разделе.* --- ### Ежедневный ритм (Дни 1–28) **Утро (08:00–09:00) — отправить квест дня:** ``` [Имя ребёнка], привет! Сегодняшний квест: 🏠 Убери стол после завтрака Вот что сделать: 1. Убери тарелки и кружки в раковину 2. Протри стол влажной салфеткой 3. Поставь стулья на место Примерно 5 минут. Когда сделаешь — напиши «Готово» 👍 ``` *Отправлять в 08:00 или когда родитель указал как удобное время.* --- **Когда ребёнок написал «Готово» — уведомление родителю:** ``` [Имя] говорит, что сделал: «Убери стол после завтрака». Подтверди 👍 или напиши, если нужно уточнить. ``` --- **Когда родитель подтвердил — ответ ребёнку:** ``` Засчитано ✅ Ключики: [N] / 5 [Если накопил 5 — добавить строку:] 🗝️🗝️🗝️🗝️🗝️ Барабан разблокирован! Скажи маме / папе — пора крутить! ``` --- **Если родитель не подтвердил за 24 часа — авто-ответ ребёнку:** ``` Засчитано автоматически ✅ (папа/мама были заняты) Ключики: [N] / 5 ``` *Это критически важно: стрик ребёнка не должен зависеть от забывчивости родителя.* --- **Если ребёнок не написал «Готово» до 20:00 — напоминание:** ``` [Имя], ещё не поздно. Квест сегодня: [название] Готово? ``` *Только одно напоминание в день. Не давить.* --- ### Еженедельный ритм **Каждое воскресенье (19:00) — Weekly Report родителю:** Формат: текстовое сообщение в Telegram. Позже — короткое Loom-видео (опционально). ``` 📋 Итоги недели [имени] ✅ Выполнено квестов: [N] из [N] 🔥 Стрик: [N] дней 🗝️ Ключиков накоплено: [N] 🎡 Барабан покрутил: [да/нет] → [выпал приз] Что растёт: [Категория навыка] → [конкретное наблюдение] Пример: «Убирает стол после еды — 5 дней из 7. Неделю назад делал только с напоминанием.» Что стоит добавить на следующей неделе: [Одна рекомендация] Пример: «Попробуйте добавить квест в учебной категории — бытовые навыки уже идут хорошо.» До следующего отчёта 👋 ``` --- ### Барабан — ручная механика Когда ребёнок накопил 5 ключиков — «запустить барабан» вручную. **Алгоритм:** 1. Открыть Google Sheet семьи → список желаний с редкостью 2. Запустить взвешенную случайную выборку вручную: ```python # Простой скрипт на Python (запустить локально): import random wishes = [ {"name": "Шоколадка", "rarity": "common", "weight": 70}, {"name": "Поход в кино", "rarity": "rare", "weight": 25}, {"name": "День с папой без телефона", "rarity": "epic", "weight": 5}, ] pool = [] for wish in wishes: pool.extend([wish] * wish["weight"]) result = random.choice(pool) print(result["name"], "-", result["rarity"]) ``` *Альтернатива без кода: написать каждое желание на бумажку пропорционально весу (70 обычных, 25 редких, 5 эпических), свернуть и вытащить случайно.* 3. Отправить ребёнку: ``` 🎡 БАРАБАН КРУТИТСЯ... ... 🎁 Тебе выпало: [название приза]! [Обычный: ⚪] / [Редкий: 🟡] / [Супер-редкий: 🔴] Папа/мама уже знают 🎉 ``` 4. Отправить родителю: ``` 🎁 [Имя] выиграл: [название приза] Редкость: [обычный / редкий / супер-редкий] Что нужно сделать: [описание] Когда реализуешь — напиши, мы запишем в историю. ``` 5. Обновить Google Sheet: спин, выпавший приз, дата. --- ## Шаблоны квестов по категориям ### 🏠 Бытовые навыки (Household) **L3 — Новичок (подробная инструкция):** ``` Убери за собой после ужина Вот что сделать: 1. Отнеси свою тарелку, стакан и столовые приборы в раковину 2. Ополосни их водой 3. Протри место, где сидел, влажной салфеткой 4. Задвинь стул Примерно 5 минут. ``` **L2 — Знакомо (чеклист):** ``` Убери за собой после ужина ☐ Посуда в раковину ☐ Стол протёрт ☐ Стул задвинут ``` **L1 — Умеет (только цель):** ``` Убери за собой после ужина. ``` --- **Банк квестов — Household:** | Квест | L3 / L2 / L1 | Мин. возраст | |---|---|---| | Убрать за собой после еды | L3 | 7 | | Застелить постель | L3 | 7 | | Собрать и вынести мусор | L2 | 8 | | Протереть пыль в своей комнате | L2 | 8 | | Загрузить посуду в посудомойку | L2 | 9 | | Пропылесосить свою комнату | L2 | 9 | | Приготовить простой завтрак | L3 | 10 | | Постирать свои вещи | L3 | 11 | --- ### 📚 Учебные навыки (Academic) | Квест | L3 / L2 / L1 | |---|---| | Сделать домашнее задание до ужина | L2 | | Подготовить портфель на завтра вечером | L3 | | Прочитать 10 страниц любой книги | L1 | | Повторить таблицу умножения (5 мин) | L3 | | Записать план на завтра (3 пункта) | L2 | --- ### ❤️ Эмоциональные навыки (Emotional) | Квест | L3 / L2 / L1 | |---|---| | Сказать кому-то из семьи «спасибо» за конкретное дело | L3 | | Написать в дневник: как прошёл день (3 предложения) | L2 | | Рассказать родителю о чём-то, что понравилось сегодня | L1 | | Помочь кому-то в семье без просьбы | L2 | --- ### 💪 Физические навыки (Physical) | Квест | L3 / L2 / L1 | |---|---| | Сделать зарядку (10 мин) — с инструкцией | L3 | | Сходить на прогулку (20 мин) | L1 | | 20 отжиманий / приседаний (любых, как умеет) | L2 | --- ### 👥 Социальные навыки (Social) | Квест | L3 / L2 / L1 | |---|---| | Позвонить бабушке / дедушке самостоятельно | L2 | | Познакомиться с кем-то новым (в школе, на кружке) | L3 | | Спросить у продавца в магазине что-нибудь самому | L3 | --- ## Google Sheet — Мастер-файл для всех семей **Структура файла:** ``` Лист 1: СВОДКА | Семья | Неделя | Квесты сделано | Квесты всего | Completion % | Ключики | Стрик | Барабан | Приз реализован | |-------|---------|---------------|-------------|--------------|---------|-------|---------|-----------------| Лист 2: СЕМЬЯ [N] | Дата | Квест | Категория | ZPD уровень | Статус (Сделано/Пропущено/Авто) | Ключики накоплено | Стрик | Заметка | Лист 3: БАРАБАН | Семья | Дата спина | Выпал приз | Редкость | Реализован (да/нет) | Дата реализации | Лист 4: ЖЕЛАНИЯ | Семья | Желание | Редкость | Измерение (деньги/время/эмоции) | Активно (да/нет) | ``` --- ## Нагрузка на основателя Честная оценка по времени при 10–15 семьях: | Задача | Время/день | |---|---| | Отправить квесты (10–15 сем.) | 20–30 мин | | Обработать «Готово» и уведомить родителей | 10–15 мин | | Следить за авто-подтверждениями | 5 мин | | Обновить Google Sheet | 10 мин | | **Итого в рабочие дни** | **45–60 мин** | | Weekly Report × 15 семей | 2–3 ч (воскресенье) | | Барабан (когда выпадает) | 10 мин/событие | **Итого в неделю: ~8–10 часов.** Это реалистичная нагрузка для основателя на 4 недели теста. После подтверждения W4 retention — строить автоматизацию. --- ## Что делать, если что-то идёт не так ### Ребёнок перестал отвечать (3+ дня тишины) ``` [Имя ребёнка], всё в порядке? Квесты тебя ждут. Когда будешь готов — просто напиши. Без спешки. ``` *Одно сообщение. Не давить. Если тишина продолжается — позвонить родителю:* ``` «Хочу убедиться, что всё в порядке. Что происходит — ему неинтересно, или что-то случилось?» ``` --- ### Родитель не подтверждает квесты 2+ дня подряд *Написать родителю (не ребёнку):* ``` [Имя], у [ребёнка] накопилось [N] неподтверждённых квестов. Я зачислил автоматически — стрик цел. Но если хочешь видеть реальную картину в отчёте — лучше подтверждать вручную. Это занимает 10 секунд: просто «✅» в ответ на моё сообщение. ``` --- ### Ребёнок говорит, что квест слишком лёгкий / скучный *Зафиксировать. Это сигнал для ZPD-сдвига. Повысить уровень:* ``` «Понял! Значит, для тебя это уже легко. Даю квест сложнее — посмотрим, как справишься.» ``` *Обновить ZPD-уровень в таблице семьи.* --- ### Ребёнок говорит, что квест слишком сложный *Не переделывать молча. Сказать открыто:* ``` «Хорошо, что сказал. Это значит, нужно начать с чего-то проще. Вот версия для начинающих: [упрощённый квест]. Когда освоишь — перейдём на следующий уровень.» ``` --- ### Родитель не реализовал приз (прошло 3+ дня) *Написать родителю — мягко:* ``` [Имя], [ребёнок] выиграл [приз] [N] дней назад. Он ещё не упоминал? Если что-то помешало — дай знать, я подберу замену или подберём другое время. ``` --- ## Признаки того, что тест идёт хорошо - Ребёнок пишет «Готово» без напоминания - Родитель сам спрашивает «когда следующий квест?» - Ребёнок упоминает барабан в разговоре с родителем - Родитель пересылает Weekly Report кому-то в семье - Семья сама предлагает идеи для новых квестов --- ## Признаки того, что что-то идёт не так - Ребёнок стабильно не отвечает 2+ дня - Completion rate < 40% за неделю - Родитель не подтверждает и не объясняет почему - Приз выпал, но родитель «забыл» реализовать 5+ дней *При любом из этих сигналов — позвонить семье в течение 24 часов. Молчание = потеря семьи.* --- ## Конец теста — W4 звонок На 28-й день — финальный звонок по протоколу из W4 Retention Test Protocol. После звонка: 1. Заполнить сводную таблицу: completion rate, streak, барабан, NPS, готовность платить 2. Посчитать W4 retention: кол-во семей с ≥1 квестом на неделе 4 / общее кол-во 3. Принять решение по дереву диагностики (см.
