# Levera

Levera is a family product concept for helping children aged 8–12 build practical skills through calibrated quests, visible progress, and parent participation.

## Repository status

This repository currently contains product, research, validation, business, marketing, technical, and legal documents. It does **not yet contain application source code**.

## Structure

- `product/` — canonical product definition, roadmap, UX, onboarding.
- `research/` — competitors, market evidence, audience and raw research.
- `validation/` — interview and experiment protocols.
- `technical/` — technical specifications.
- `brand/` — brand identity.
- `marketing/` — acquisition, content, store and PR materials.
- `business/` — business model, pricing, partnerships and fundraising.
- `legal/` — legal drafts requiring professional review.
- `archive/` — superseded or incomplete material.

## Current priority

Validate the core loop with real families before building the full application:

`quest → completion → parent confirmation → key → wish wheel → fulfilled reward → weekly report`

Target audience for the first pilot: Russian-speaking families with children aged 8–12.
