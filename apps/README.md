# Applications

The first implementation is Levera Personal for Pavel's own use.

Planned structure:

```text
apps/
  mobile/   React Native / Expo personal development application
```

The active specification is `../docs/product/PERSONAL-FIRST-DIRECTION.md`.
The live build plan is `../docs/EXECUTION-PLAN.md`.

The initial application is local-first and should provide:

- goals and skills;
- a visual skill tree;
- quests with adjustable support levels;
- evidence, reflection and progress history;
- spaced repetition;
- weekly review;
- Markdown import/export for second-brain integration.

Family administration, payments, AI quest generation and automatic two-way Obsidian synchronization are deferred.
