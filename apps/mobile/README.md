# Levera Personal mobile

Local-first React Native / Expo application.

## Commands

```bash
npm install
npm run start
npm run typecheck
npm test
```

## Current foundation

- Expo Router tab shell
- Levera design tokens
- Personal domain model
- SQLite schema and migration runner
- Demo Today, Skills, Progress, Review and Settings screens
- Pure domain tests for completion XP and skill-health derivation

The current UI uses demo data. The next vertical slice replaces it with SQLite-backed goal, skill and quest creation.
